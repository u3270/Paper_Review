## 1. H-WPS: Hybrid Wireless Positioning System Using an Enhanced Wi-Fi FTM/RSSI/MEMS Sensors Integration Approach

- **저자**: Yue Yu, Ruizhi Chen, Liang Chen, Wei Li, Yuan Wu, Haitao Zhou
- **출처**: IEEE Internet of Things Journal, Vol. 9, Issue 14, pp. 11827–11842
- **출판일**: 2022-07-15
- **DOI**: [10.1109/jiot.2021.3132023](https://doi.org/10.1109/jiot.2021.3132023)
- **관련도 등급**: A

**요약**: 이 논문은 Wi-Fi Fine Timing Measurement(FTM), RSSI 기반 크라우드소싱 지문 인식, MEMS 관성 센서를 결합한 하이브리드 실내 측위 시스템(H-WPS)을 제안한다. 경량 보행자 관성 항법(PINS)에 다중 제약 조건과 전역 최적화 모델을 적용하여 INS 갱신에서 발생하는 누적 오차를 줄인다. 또한 크라우드소싱 궤적을 활용해 Wi-Fi 지문 데이터베이스를 생성하는 딥러닝 프레임워크를 개발하고, 세 가지 다중 소스 융합 모델로 PINS, Wi-Fi FTM, RSSI 지문 정보를 결합하여 실시간으로 Wi-Fi 레인징 바이어스를 보정한다. 실험 결과 기존 단일 소스 위치 추정 방식보다 정밀하고 범용적인 실내 측위 성능을 달성했으며 미터 수준의 정확도를 확인했다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: Wi-Fi FTM 레인징 바이어스의 실시간 보정 기법은 RFSoC 기반 광대역 시스템에서 관측되는 지연 추정 오차 모델링과 직접적으로 연관되며, 다중 센서 융합을 통한 레인징 정확도 개선 접근은 향후 레인징 오차 모델 설계에 참고할 수 있다.

**잠재적으로 관련 있는 논문 섹션**: Ranging Error Model, Related Work

## 2. Robust WiFi Channel State Information-Based Localization via Sequence Time Alignment

- **저자**: Dongwoo Kang, Seho Park, Taehyeon Kim
- **출처**: 2025 IEEE/IEIE International Conference on Consumer Electronics-Asia (ICCE-Asia)
- **출판일**: 2025-10-27
- **DOI**: [10.1109/icce-asia67487.2025.11263593](https://doi.org/10.1109/icce-asia67487.2025.11263593)
- **관련도 등급**: B

**요약**: 이 논문은 다중 ESP 장치에서 수집한 CSI(Channel State Information) 스트림의 노이즈, 환경 변화, 비동기 패킷 도착 문제를 해결하기 위한 전처리 기반 프레임워크를 제안한다. 진폭 추출, Butterworth 필터링, 새로운 시간 정렬 동기화 전략을 통합하고, SVM 분류기를 이용해 근/원거리 및 좌/우 위치 분류 실험을 수행했다. 동기화 없이는 78.5%였던 정확도가 제안된 2초 창 시간 정렬 전처리를 통해 95.5%로 향상되었다. 저자들은 정교한 전처리가 경량 모델의 CSI 기반 측위 강건성을 크게 높이는 핵심 설계 요소임을 강조한다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 다중 수신 장치 간 CSI 시퀀스 시간 정렬 기법은 광대역 OFDM 시스템에서 다중 안테나/다중 채널 지연 추정 시 동기화 오차를 줄이는 문제와 방법론적으로 연결되며, CSI 기반 위치 추정의 전처리 파이프라인 설계에 참고할 수 있다.

**잠재적으로 관련 있는 논문 섹션**: System Model, Experiment

## 3. Millimeter-Wave Digital Beamforming Receiver Using RFSoC FPGA for MIMO Communications

- **저자**: Kefayet Ullah, Satheesh Bojja Venkatakrishnan, John L. Volakis
- **출처**: 2022 IEEE 22nd Annual Wireless and Microwave Technology Conference (WAMICON)
- **출판일**: 2022-04-27
- **DOI**: [10.1109/wamicon53991.2022.9786120](https://doi.org/10.1109/wamicon53991.2022.9786120)
- **관련도 등급**: B

**요약**: 이 논문은 단일 ADC를 사용하는 디지털 빔포머를 제안하여 MIMO 응용에서 채널 간 간섭(ICI) 문제를 해결한다. 핵심은 RFSoC(Radiofrequency System-on-Chip) FPGA에서 생성한 직교 Walsh-Hadamard 부호를 이용한 코드 다중화 방식으로, 최대 400MHz 클럭 주파수에서 동작한다. 다중화된 신호는 단일 채널을 통해 디지털화되며, UWB 밀착 결합 다이폴 배열(TCDA)과 직접 변환 수신기 구조를 이용해 실험적으로 검증되었다. 측정 결과 SNR 저하 없이 36dB의 ICI 제거 성능을 달성했으며, 저자들은 이것이 현재까지 보고된 가장 높은 클럭 주파수 및 ICI 값이라고 밝힌다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 단일 ADC 기반 RFSoC FPGA 디지털 빔포밍 구조는 광대역 UWB 신호 처리를 위한 하드웨어 구현 방식을 제시하며, RFSoC 플랫폼에서의 고클럭 코드 다중화 기법은 향후 wideband OFDM 레인징 시스템의 RF 프론트엔드 설계에 참고할 수 있다.

**잠재적으로 관련 있는 논문 섹션**: System Model, Experiment

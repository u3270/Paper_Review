## 1. Secure Wi-Fi Ranging Today: Security and Adoption of IEEE 802.11az/bk

- **저자**: Nikola Antonijević, Bernhard Etzlinger, Dave Singelée, Bart Preneel
- **출처/venue**: arXiv.org (preprint)
- **발행일**: 2026-03-19
- **DOI**: 10.48550/arxiv.2603.18687
- **Publisher URL**: https://arxiv.org/abs/2603.18687
- **관련도 등급**: A
- **Zotero 상태**: 기존 항목과 중복 확인됨 (item key `6TAZS3HR`, 태그 `daily-research`/`relevance-A`/`wifi-ranging` 이미 부착) → 신규 추가 생략

**요약**: 이 논문은 IEEE 802.11az(Fine Timing Measurement 보안 강화)와 802.11bk(대역폭 확장을 통한 UWB급 정밀도 목표) 표준을 보안 및 실제 배포 관점에서 분석한다. 표준 분석과 상용/개발용 하드웨어에서의 시뮬레이션·측정을 결합하여, 논리 계층에서는 인증되지 않은 레인징, 다운그레이드 공격, 서비스 거부 공격이 가능함을 보이고, 물리 계층에서는 보안 레인징 파형의 예측 가능성과 심볼 반복이 스펙트럼 마스크 준수에 미치는 영향을 다룬다. 결과적으로 안전한 Wi-Fi 레인징은 설정 선택에 매우 민감하며 기존 하드웨어에서 구현이 까다롭다는 점을 보여주고, 벤더 및 표준화 기구를 위한 실무 지침을 제시한다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 802.11az/bk는 사용자의 RFSoC 기반 광대역 Wi-Fi 레인징 연구가 직접 대상으로 하는 최신 표준이며, 이 논문이 지적하는 파형 설계·대역폭·타이밍 취약점은 향후 시스템 설계에서 고려해야 할 실질적 제약 조건을 제공한다.

**관련 가능 섹션**: Introduction, Related Work

## 2. High-Accuracy Wi-Fi Ranging via Joint FTM-CSI Processing and Super-Resolution ToF Estimation

- **저자**: Shikun Yao, Jian Sun, Feiyu Wang, Hengtai Chang, Wensheng Zhang, Cheng-Xiang Wang
- **출처/venue**: 2025 IEEE 25th International Conference on Communication Technology (ICCT)
- **발행일**: 2025-10-16 (Crossref/Zotero 메타데이터 기준; candidates.json 원본의 "2016-10-18" 표기는 발행연도 2025와 불일치하여 오류로 판단, 사용하지 않음)
- **DOI**: 10.1109/icct67417.2025.11374065
- **Publisher URL**: https://doi.org/10.1109/icct67417.2025.11374065
- **관련도 등급**: A
- **Zotero 상태**: 신규 추가 완료 (item key `3WF843M8`), 태그 `daily-research`/`wifi-ranging`/`relevance-A` 부착. 오픈액세스 PDF 없음 확인되어 첨부하지 않음(페이월 자동 다운로드 금지 원칙 준수)

**요약**: 이 논문은 Wi-Fi FTM(Fine Time Measurement)이 다중경로 및 NLoS 환경에서 성능이 저하되는 문제를 CSI(Channel State Information) 기반 ToF(Time of Flight) 추정으로 보정하는 방법을 제안한다. 직접 경로 ToF 추정을 위해 두 가지 알고리즘, 즉 Expectation-Maximization과 Nelder-Mead 심플렉스 기법을 결합한 InGEM(Iterative non-Grid EM)과, 계산 복잡도를 낮춘 서브스페이스 기반 FD/UMUSIC(Frequency Decimated/Unitary-MUSIC)을 제시한다. 시뮬레이션과 실내 실험에서 두 방법 모두 80% 사례에서 서브미터 수준의 정확도를 달성하여 FTM 단독 대비 성능을 크게 개선함을 보였다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: FTM과 CSI를 결합한 지연 보정 및 UMUSIC 기반 초해상도 ToF 추정 기법은 사용자의 연구 우선순위(Wi-Fi 레인징, MUSIC/초해상도 지연 추정)와 직접적으로 일치하며, RFSoC 플랫폼에서 실시간 CSI 처리 및 지연 추정 알고리즘 구현 시 참고할 수 있는 구체적 알고리즘 설계(InGEM, FD/UMUSIC)를 제공한다.

**관련 가능 섹션**: Delay Estimation, Ranging Error Model

## 3. Crowdsourcing Ubiquitous Indoor Localization with Non-Cooperative Wi-Fi Ranging

- **저자**: Emerson (E.M.S.L.) Sie, Enguang Fan, Federico Cifuentes-Urtubey, Deepak Vasisht
- **출처/venue**: arXiv.org (preprint)
- **발행일**: 2025-06-23 (Zotero 기록 기준; candidates.json에는 2025-06-22로 기재, arXiv 버전 갱신에 따른 하루 차이로 추정)
- **DOI**: 10.48550/arxiv.2506.18317
- **Publisher URL**: https://arxiv.org/abs/2506.18317
- **관련도 등급**: A
- **Zotero 상태**: 기존 항목과 중복 확인됨 (item key `U6IKJNFV`, 태그 `daily-research`/`relevance-A`/`wifi-ranging` 이미 부착) → 신규 추가 생략. 동일 연구의 포스터 버전("Poster: Scalable Indoor Localization with Non-Cooperative Wi-Fi Ranging", DOI 10.1145/3680207.3765680)도 후보에 있었으나 본 전체 논문과 중복 내용으로 판단하여 오늘 선정에서 제외함.

**요약**: 이 논문은 PeepLoc이라는 확장 가능한 Wi-Fi 기반 실내 측위 시스템을 제안하며, 기존 인프라와 수정되지 않은 모바일 기기만으로 동작한다. 비협조적(non-cooperative) 방식으로 임의의 AP로부터 ToF(time-of-flight)를 추출하고, 보행자 추측항법(PDR)과 크라우드소싱을 이용해 AP를 앵커로 부트스트래핑하는 기법을 결합한다. 4개 건물에서 상용 하드웨어로 검증한 결과 평균 3.41m, 중앙값 3.06m의 오차를 달성하여 기존 상용 실내 측위 시스템보다 우수하고 실외 GPS 수준에 근접함을 보였다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 비협조적 ToF 추출과 크라우드소싱 기반 앵커 보정 기법은 실제 배포 환경에서의 Wi-Fi 레인징 실용성 문제를 다루며, RFSoC 기반 시스템의 실외/대규모 실내 배포 시나리오 설계에 참고할 수 있는 시스템 레벨 접근을 제공한다.

**관련 가능 섹션**: System Model, Experiment

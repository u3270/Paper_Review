## 1. Locating Smartphones Indoors Using Built-In Sensors and Wi-Fi Ranging With an Enhanced Particle Filter

- **저자**: Shihao Xu, Ruizhi Chen, Yue Yu, Guangyi Guo, Lixiong Huang
- **출처**: IEEE Access
- **발행일**: 2019-07-18
- **DOI / URL**: 10.1109/access.2019.2927387 / https://doi.org/10.1109/access.2019.2927387
- **관련도 등급**: A

**요약**: IEEE 802.11-2016 표준에 도입된 Wi-Fi FTM(Fine Timing Measurement) 프로토콜을 활용해 실내 스마트폰 위치를 추정하는 연구입니다. 관성센서 기반 PDR(보행자 추측항법)과 Wi-Fi FTM 레인징을 융합하기 위해 두 가지 상태 갱신 전략과 발산 감지·재초기화 기준을 갖춘 향상된 파티클 필터(PF)를 제안합니다. 적응형 기울기 보정으로 PDR의 실시간 헤딩 추정 정확도를 높이고, PDR 변위 추정을 이용해 Wi-Fi FTM 이상치를 검출합니다. 실험 결과 동적 실내 환경에서 86.7%의 경우 1m 이내 오차를 달성했으며, 파티클 2000개 기준 평균 계산 시간은 0.5초 미만입니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: FTM 기반 레인징이 실제 실내 환경에서 겪는 이상치·발산 문제와 이를 보정하는 센서 융합 기법을 제시하므로, 레인징 오차 모델링과 실측 검증 단계에서 참고할 수 있습니다. 다만 핵심 기여는 지연 추정 자체가 아니라 상위 레벨의 위치 추정 필터이므로 직접적인 지연 추정 기법과는 다소 거리가 있습니다.

- **관련 섹션**: Ranging Error Model / Experiment
- **권장 조치**: Save for later

## 2. Non-cooperative wi-fi localization & its privacy implications (Wi-Peep)

- **저자**: A. Abedi, Deepak Vasisht
- **출처**: ACM/IEEE International Conference on Mobile Computing and Networking (MobiCom) 2022
- **발행일**: 2022-10-14
- **DOI / URL**: 10.1145/3495243.3560530 / https://doi.org/10.1145/3495243.3560530
- **관련도 등급**: A

**요약**: 802.11 프로토콜의 허점을 이용해 접근 권한이 없는 Wi-Fi 기기로부터 응답을 유도하고, 새로운 time-of-flight(ToF) 측정 기법으로 위치를 추정하는 Wi-Peep 시스템을 제안합니다. 대상 기기에 별도의 하드웨어·소프트웨어 수정이나 접근 권한 없이도 위치 추정이 가능하며, 20달러 미만·10g 이하의 저가 장치를 드론에 탑재해 건물 여러 층에 걸쳐 미터급 정확도로 Wi-Fi 기기 위치를 실증했습니다. 비협조적(non-cooperative) 상황에서도 ToF 기반 거리 추정이 가능함을 보여주는 동시에, 이로 인한 보안·프라이버시 취약점과 완화 기법도 함께 논의합니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 비협조적 환경에서의 ToF 추출 기법은 FTM 프로토콜의 취약점과 대안적 타이밍 측정 방식에 대한 통찰을 제공합니다. RFSoC 기반 레인징 시스템 설계 시 타이밍 정밀도의 한계와 보안 위협 모델을 함께 고려하는 데 직접적으로 참고할 가치가 있습니다.

- **관련 섹션**: Related Work / Delay Estimation

## 3. Virtual Antenna Array and Fractional Fourier Transform-Based TOA Estimation for Wireless Positioning

- **저자**: Zhigang Chen, Lei Wang, Mengya Zhang
- **출처**: Sensors (MDPI)
- **발행일**: 2019-02-01
- **DOI / URL**: 10.3390/s19030638 / https://doi.org/10.3390/s19030638
- **관련도 등급**: A

**요약**: OFDM 기반 WLAN 무선측위를 위해 가상 안테나 배열과 분수 푸리에 변환(FRFT)을 결합한 2차원 초분해능 TOA 추정 알고리즘을 제안합니다. 이동 경로 상 등간격 위치에서 얻은 채널 주파수 응답(CFR)들을 가상 안테나 배열로 간주하고, 다중경로의 위상 변화를 기존의 선형 근사 대신 처프(chirp) 형태의 2차 함수로 모델링하여 큰 가상 배열 크기에서도 정확도를 유지합니다. FRFT 영역에서 처프 형태 다중경로 성분들이 에너지 집중 특성을 갖는 점을 이용해 이들을 분리한 후, 기존 주파수영역 TOA 추정 기법을 적용하여 다중경로 해상도와 TOA 추정 성능을 개선합니다. 시뮬레이션 결과 실제 안테나 배열 없이도 우수한 초분해능 TOA 추정 성능을 확인했습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 광대역 OFDM 신호에서의 다중경로 분리 및 초분해능 지연 추정을 정면으로 다루는 방법론으로, RFSoC 기반 광대역 Wi-Fi 레인징 시스템의 지연 추정 알고리즘 설계 및 MUSIC/SAGE 계열 기법과의 비교 연구에 직접적으로 참고할 가치가 큽니다.

- **관련 섹션**: Delay Estimation / System Model

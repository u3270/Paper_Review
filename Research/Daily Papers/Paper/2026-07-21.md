# 2026-07-21 연구 논문 다이제스트

## 1. ConFi: Convolutional Neural Networks Based Indoor Wi-Fi Localization Using Channel State Information

- **저자**: Hao Chen, Yifan Zhang, Wei Li, Xiaofeng Tao, Ping Zhang
- **출처**: IEEE Access (2017)
- **발행일**: 2017-07-21
- **DOI**: 10.1109/access.2017.2749516
- **URL**: https://ieeexplore.ieee.org/document/8027020/
- **관련도 등급**: A

**요약**: 이 논문은 CSI(Channel State Information)를 시간-주파수 행렬 형태의 이미지로 변환하여 CNN(합성곱 신경망)으로 실내 Wi-Fi 위치추정을 수행하는 ConFi 기법을 제안합니다. 기존 RSSI 기반 방식보다 위치 관련 정보를 더 풍부하게 담고 있는 CSI를 3개의 합성곱 층과 2개의 완전연결 층으로 구성된 5층 CNN에 입력하여 분류 문제로 모델링합니다. 오프라인 단계에서 참조점(RP)의 CSI로 학습하고, 온라인 단계에서는 출력값이 높은 RP들의 가중 중심으로 위치를 추정합니다. 다양한 실험을 통해 기존 기법 대비 우수한 위치추정 성능을 입증했습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: CSI를 이미지 형태로 구조화하여 딥러닝에 활용하는 접근은 CSI 기반 지연/거리 추정 파이프라인의 특징 추출 단계에 참고할 수 있는 방법론입니다. 다만 본 논문은 분류 기반 위치추정에 초점을 맞추고 있어, 직접적인 지연 추정 알고리즘(MUSIC/SAGE 등)과는 결이 다르지만 CSI 전처리 및 데이터 표현 방식은 참고 가치가 있습니다.

**잠재적으로 관련된 논문 섹션**: Related Work, System Model

## 2. Convolutional neural networks based indoor Wi-Fi localization with a novel kind of CSI images

- **저자**: Haihan Li, Xiangsheng Zeng, Yunzhou Li, Shidong Zhou, Jing Wang
- **출처**: China Communications (2019)
- **발행일**: 2019-09-21
- **DOI**: 10.23919/jcc.2019.09.019
- **URL**: https://ieeexplore.ieee.org/document/8851471/
- **관련도 등급**: A

**요약**: 이 논문은 안테나 간 위상 차이와 부반송파별 진폭 정보를 결합하여 AoA(도래각), TOA(도래시간), 진폭 정보를 모두 포함하는 새로운 형태의 3채널 RGB CSI 이미지를 구성하는 방법을 제안합니다. 이렇게 구성된 이미지를 CNN으로 학습시켜 실내 Wi-Fi 위치추정 정확도를 향상시켰습니다. 오프라인 단계에서 학습 위치의 CSI 이미지로 CNN을 훈련하고, 온라인 단계에서 테스트 위치의 이미지를 입력해 출력값 가중 평균으로 위치를 추정합니다. 대표적인 실내 환경에서의 실험을 통해 기존 CSI 이미지 구성 방식 대비 향상된 성능을 확인했습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: TOA(도래시간) 정보를 CSI로부터 명시적으로 추출하여 위치추정에 활용한다는 점에서, 광대역 OFDM 기반 지연 추정 연구와 접점이 있습니다. 특히 위상 차이 기반 AoA 추출 기법은 MUSIC/SAGE와 같은 초분해능 지연/각도 추정 알고리즘의 전처리 단계와 비교해볼 가치가 있습니다.

**잠재적으로 관련된 논문 섹션**: Delay Estimation, System Model

## 3. Lessons From Deploying Learning-Based CSI Localization on a Large-Scale ISAC Platform

- **저자**: Tianyu Zhang, Dongheng Zhang, Ruixu Geng, Xuecheng Xie, Shuai Yang, Yan Chen
- **출처**: IEEE Internet of Things Journal
- **발행일**: 정확한 발행일 메타데이터가 소스 간 불일치(원본 레코드상 2015-01-15, 발행연도 2026으로 표기됨) — 메타데이터만으로 판단했으며 정확한 날짜는 출판사 페이지에서 재확인 필요
- **DOI**: 10.1109/jiot.2025.3633282
- **URL**: https://ieeexplore.ieee.org/document/11250670/
- **관련도 등급**: A

**요약**: 이 논문은 400개 이상의 액세스 포인트(AP)를 갖춘 실제 대형 건물에서 ISAC(통신-센싱 통합) 패러다임 하에 대규모 CSI 기반 실내 위치추정 시스템을 배포한 경험을 다룹니다. 기존 CSI 기반 연구가 소규모 통제 환경에 국한되어 일반화가 어려웠던 한계를 지적하며, 비표지(unlabeled) 데이터의 미활용과 CSI 측정의 이질성이라는 두 가지 핵심 문제를 제시합니다. 그래프 기반 구조로 이기종 CSI 데이터를 모델링하고, 공간·시간적 사전지식을 반영한 프리텍스트 사전학습과 신뢰도 인식 미세조정 전략을 도입했습니다. 5개 층, 25,600㎡ 규모의 실험에서 중간값 위치 오차 2.17m, 층 정확도 99.49%를 달성하여 기존 최고 성능 대비 평균절대오차(MAE)를 18.7% 개선했습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 대규모 실환경에서의 CSI 이질성 문제 및 ISAC 배포 경험은, RFSoC 기반 광대역 시스템을 실제 다중 AP 환경에 확장할 때 마주칠 수 있는 실용적 난제(측정 이질성, 비표지 데이터 활용)에 대한 시사점을 제공합니다. 다만 본 논문은 학습 기반 분류/회귀 위치추정에 중점을 두고 있어, 물리 계층의 지연 추정 알고리즘과는 직접적인 연관성이 제한적입니다.

**잠재적으로 관련된 논문 섹션**: Experiment, Related Work

## 1. Wi-Fi Ranging in Mixed LOS/NLOS Environments via Channel Statistics-Guided Learning

- **저자**: Jian Wang, Lixing Wang, Qingxu Deng
- **출처**: 2025 11th International Conference on Computer and Communications (ICCC)
- **발행일**: 2025-12-12
- **DOI / URL**: 10.1109/iccc68654.2025.11438229 / https://doi.org/10.1109/iccc68654.2025.11438229
- **관련도 등급**: A

**요약**: 상용 Wi-Fi 기기로 수집한 실측 데이터를 바탕으로 LOS/NLOS가 혼재된 실내 환경에서 FTM(Fine Timing Measurement) 레인징 오차 문제를 다룹니다. 채널 임펄스 응답(CIR)의 RMS 지연 확산(RMS-DS)이 클수록 FTM 레인징 오차가 커진다는 강한 양의 상관관계를 실험적으로 규명했으며, CSI에서 추출한 라이시안(Rician) K-factor 역시 오차 보정에 기여함을 보였습니다. 이를 바탕으로 CSI를 주 입력으로, RMS-DS와 K-factor를 보조 입력으로 사용하는 보정 모델을 제안합니다. ESP32-C6 플랫폼에서 CSI와 FTM 데이터를 동시에 수집해 검증한 결과, 레인징 RMSE를 2.26m에서 0.41m로 약 82% 감소시켰습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: FTM 기반 레인징 오차를 채널 통계(지연 확산, K-factor)로 설명하고 보정하는 접근은 RFSoC 기반 광대역 시스템에서 다중경로 환경에 따른 레인징 오차 모델을 구축하고 검증하는 데 직접적으로 참고할 수 있습니다. 추가 하드웨어 없이 CSI만으로 보정한다는 점도 실측 검증 설계에 시사점을 줍니다.

- **관련 섹션**: Ranging Error Model / Experiment
- **권장 조치**: Read now

## 2. Self-Calibrated Multi-Floor Localization Based on Wi-Fi Ranging/Crowdsourced Fingerprinting and Low-Cost Sensors

- **저자**: Qiao Wan, Xiaoqi Duan, Yue Yu, Ruizhi Chen, Liang Chen
- **출처**: Remote Sensing (MDPI), Vol. 14, Issue 21, p.5376
- **발행일**: 2022-10-27
- **DOI / URL**: 10.3390/rs14215376 / https://doi.org/10.3390/rs14215376
- **관련도 등급**: B

**요약**: Wi-Fi 레인징, 크라우드소싱 지문(fingerprinting), 저가 센서를 결합한 자가보정형 다층(multi-floor) 실내 측위 프레임워크(SM-WRFS)를 제안합니다. 헤딩·고도 바이어스, 보폭 스케일 계수, Wi-Fi 레인징 바이어스 등 측위 파라미터를 자율적으로 보정하여 3차원 위치 추정 정확도를 높였습니다. 후방 스무딩 알고리즘과 딥러닝 모델을 이용해 QR코드 랜드마크 기반 크라우드소싱 CSI/Wi-Fi 지문 데이터베이스를 자동 구축하고, 적응형 확장 칼만 필터로 여러 위치 정보원을 융합합니다. 실측 실험에서 다양한 환경에 걸쳐 미터급 3차원 실내 측위 정확도를 달성했습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: 핵심 기여가 지연 추정 자체보다는 상위 레벨의 다중 센서 융합·측위 파이프라인에 있어 직접적 관련성은 제한적이지만, Wi-Fi 레인징 바이어스를 자율적으로 보정하는 절차는 RFSoC 레인징 시스템의 시스템적 오차 보정 및 실환경 검증 단계에서 참고할 가치가 있습니다.

- **관련 섹션**: Ranging Error Model / Experiment
- **권장 조치**: Save for later

## 3. An Indoor Wi-Fi Localization Algorithm Using Ranging Model Constructed With Transformed RSSI and BP Neural Network

- **저자**: Yiruo Lin, Kegen Yu, Lianxiao Hao, Jin Wang, Jinwei Bu
- **출처**: IEEE Transactions on Communications, Vol. 70, Issue 3, pp.2163–2177
- **발행일**: 2022-03
- **DOI / URL**: 10.1109/tcomm.2022.3145408 / https://doi.org/10.1109/tcomm.2022.3145408
- **관련도 등급**: B

**요약**: 수신신호강도(RSSI) 변동으로 인한 실내 Wi-Fi 측위 성능 저하를 완화하기 위한 연구입니다. 각 기준점에서 수집한 RSSI를 평행이동·스케일링으로 변환한 뒤, 유전 알고리즘(GA)으로 가중치·바이어스 초기값을 최적화한 BP(오차역전파) 신경망을 이용해 거리-신호 관계를 모델링하는 레인징 모델(GTBPD)을 제안합니다. 이후 순차 이차계획법(SQP)과 결합한 위치 추정 알고리즘(GTBPD-LSQP)을 구성하여, 서로 다른 건물의 세 개 실내 공간에서 기존 네 가지 알고리즘 대비 더 높은 측위 정확도를 실험적으로 입증했습니다.

**RFSoC 기반 광대역 Wi-Fi 레인징 연구와의 관련성**: RSSI 기반 레인징 모델로, 지연 추정이나 광대역 OFDM 신호를 직접 다루지는 않아 RFSoC 기반 연구와의 직접적 연관성은 낮습니다. 다만 신경망 기반 레인징 모델 보정과 최적화 기법은 레인징 오차 모델링 방법론 비교 관점에서 참고할 수 있습니다.

- **관련 섹션**: Ranging Error Model / Related Work
- **권장 조치**: Skip
